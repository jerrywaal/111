总结:

1. 功能需求来源于生产一线设计师；
2. 针对传统设计绘图的痛点及难点进行研究、开发；
3. 引入信息参数化技术，通过数学算法和计算机编程技术；
4. 实现参数化布置、标注自动化、统计自动化、生成自动化；
5. 致力于大幅度减少设计制图中大量重复性、机械性操作；
6. 达到设计成果的规范和标准统一，同时大大的提升了图纸质量和工作效率；
7. 顺应“数字化—信息化—智能化—智慧化”的发展趋势；
